# Speech To Text Service

- This project uses [Speech To Text Service from Azure](https://docs.microsoft.com/azure/cognitive-services/speech-service/speech-to-text)

## Getting started

- Create Service in the Azure Portal
- Create `.env` file

```js
OUTPUT_FILEPATH="output.txt"
INPUT_WAV_FILEPATH="./audios/xx.wav"
SERVICE_SUBSCRIPTION_KEY=""
SERVICE_REGION=""
```

- `npm run start`
